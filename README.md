Projeto desenvolvido como parte de um **Projeto Interdisciplinar acadêmico**, com o objetivo de integrar conhecimentos adquiridos ao longo do curso e aplicá-los na construção de uma aplicação funcional.

O projeto tem como foco a **aplicação prática de conceitos de programação, organização de código e desenvolvimento de sistemas**, utilizando boas práticas de desenvolvimento e versionamento de código.

---

# 📚 Sobre o Projeto

Projetos interdisciplinares são fundamentais na formação de desenvolvedores, pois permitem aplicar na prática conteúdos estudados em diferentes disciplinas.

Este projeto foi desenvolvido com o propósito de:

* Integrar diferentes conceitos de programação
* Desenvolver uma aplicação estruturada
* Praticar organização de código
* Utilizar controle de versão com Git
* Documentar um projeto real no GitHub

A iniciativa contribui para o desenvolvimento de habilidades essenciais na área de **desenvolvimento de software**.

# 📂 Estrutura do Projeto

A organização do projeto segue uma estrutura que facilita a manutenção e a leitura do código:

```
ProjetoInterdiciplinar
│
├── src/           # Código fonte da aplicação
├── assets/        # Recursos do projeto (imagens, arquivos auxiliares)
├── docs/          # Documentação adicional
└── README.md      # Documentação principal do projeto
```

Essa organização contribui para um projeto **mais limpo, escalável e profissional**.

# ⚙️ Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash

```

### 2️⃣ Acessar a pasta do projeto

```bash
cd ProjetoInterdiciplinar
```

### 3️⃣ Abrir o projeto em uma IDE

Você pode utilizar qualquer ambiente de desenvolvimento, como:

* Visual Studio
* Visual Studio Code
* IntelliJ
* Eclipse

### 4️⃣ Executar o projeto

Abra o arquivo principal da aplicação e execute o projeto pela sua IDE.
